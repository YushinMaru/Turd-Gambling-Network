## Context

This document provides a comprehensive validation checklist for ensuring stylistic consistency across all Turd Bird Universe documentation. Use this checklist during creation, modification, and review of content to maintain a unified narrative voice and presentation style.

## Document Structure

### Format Foundations

- [ ] **Document Framework**
  - [ ] File begins with appropriate title in heading level 1
  - [ ] Edition information follows title in prescribed format
  - [ ] Previous version reference is included when applicable
  - [ ] File size limitation warning is present

- [ ] **Section Organization**
  - [ ] Sections follow standard hierarchy (H1 → H2 → H3 → H4)
  - [ ] Section IDs use consistent capitalization (ALL-CAPS-WITH-HYPHENS)
  - [ ] Section progression follows logical flow
  - [ ] Standard sections appear in prescribed order

- [ ] **Content Containers**
  - [ ] Lists use consistent bullet or numbering formats
  - [ ] Tables follow standard structure with headers
  - [ ] Code blocks use appropriate language tags
  - [ ] Blockquotes are formatted consistently

- [ ] **Visual Structure**
  - [ ] Whitespace usage follows documentation standards
  - [ ] Paragraph length adheres to readability guidelines
  - [ ] Indentation is consistent throughout document
  - [ ] Line breaks are used appropriately

### Metadata Implementation

- [ ] **Header Information**
  - [ ] Edition number follows semantic versioning (x.y.z)
  - [ ] Created field includes creator identifier
  - [ ] Last Modified field includes modifier identifier
  - [ ] Date formats follow YYYY-MM-DD standard

- [ ] **Classification Markers**
  - [ ] Content type is clearly indicated
  - [ ] Priority level is marked when applicable
  - [ ] Confidentiality level is appropriately labeled
  - [ ] Narrative status indicators are present

- [ ] **Cross-Reference Data**
  - [ ] Related document section lists all direct connections
  - [ ] Version dependencies are clearly marked
  - [ ] Prerequisite content is identified
  - [ ] Supplementary materials are properly referenced

- [ ] **Footer Elements**
  - [ ] Version history follows standard format
  - [ ] References section includes all cited documents
  - [ ] Implementation notes follow standardized structure
  - [ ] Appendix materials are properly labeled

## Narrative Voice

### Tone Consistency

- [ ] **Universe Tone**
  - [ ] Content maintains Turd Bird Universe's corporate absurdist tone
  - [ ] Humor level matches content type requirements
  - [ ] Seriousness/frivolity balance is appropriate
  - [ ] Emotional tone fits document purpose

- [ ] **Content Type Appropriateness**
  - [ ] Technical documents maintain appropriate formality
  - [ ] Character documents reflect narrative style
  - [ ] Corporate documents use appropriate business tone
  - [ ] Timeline documents maintain historical perspective

- [ ] **Audience Adaptation**
  - [ ] Technical level matches intended audience
  - [ ] In-universe vs. meta documentation tone is appropriate
  - [ ] Jargon usage follows universe standards
  - [ ] Accessibility of language matches document purpose

- [ ] **Emotional Register**
  - [ ] Intensity level is appropriate to content
  - [ ] Emotional markers are consistent
  - [ ] Subjective language is appropriately calibrated
  - [ ] Dramatic elements follow stylistic guidelines

### Perspective Management

- [ ] **Narrative Distance**
  - [ ] Perspective maintains consistent distance from subject
  - [ ] Observer position is stable throughout document
  - [ ] First/second/third person usage is consistent
  - [ ] Omniscient vs. limited perspective is maintained

- [ ] **Temporal Perspective**
  - [ ] Past/present/future tense usage is consistent
  - [ ] Time references match narrative position
  - [ ] Flashbacks/flashforwards are clearly marked
  - [ ] Historical vs. current perspective is maintained

- [ ] **Objectivity Control**
  - [ ] Factual vs. subjective framing is consistent
  - [ ] Bias level matches document requirements
  - [ ] Editorial commentary is clearly marked
  - [ ] Character perspective vs. universe facts are distinguished

- [ ] **Meta-Narrative Elements**
  - [ ] Fourth-wall elements are used consistently
  - [ ] Self-referential content follows universe standards
  - [ ] Meta-commentary is appropriately marked
  - [ ] Universe awareness level is consistent

### Character Voice

- [ ] **Dialogue Authenticity**
  - [ ] Character speech matches established patterns
  - [ ] Vernacular and slang usage is character-appropriate
  - [ ] Sentence structure reflects character's thinking style
  - [ ] Vocabulary range matches character's background

- [ ] **Quote Implementation**
  - [ ] Character quotes are formatted consistently
  - [ ] Attribution follows standard format
  - [ ] Context is provided appropriately
  - [ ] Quote selection is representative of character

- [ ] **Speech Pattern Consistency**
  - [ ] Verbal tics are maintained
  - [ ] Cadence and rhythm match character's style
  - [ ] Formal/informal balance matches character
  - [ ] Cultural speech influences are consistent

- [ ] **Thought Representation**
  - [ ] Internal monologue is formatted consistently
  - [ ] Cognitive patterns match character psychology
  - [ ] Thought structure reflects personality traits
  - [ ] Emotional filtering is character-appropriate

## Language Elements

### Vocabulary Standards

- [ ] **Terminology Consistency**
  - [ ] Technical terms follow established definitions
  - [ ] Corporate language adheres to universe standards
  - [ ] Scientific terminology is used accurately
  - [ ] Specialized vocabulary is explained when necessary

- [ ] **Universe-Specific Vocabulary**
  - [ ] Turd Bird Universe terminology is used correctly
  - [ ] In-universe concepts maintain consistent labels
  - [ ] Made-up terms follow universe linguistic patterns
  - [ ] Unique terminology is consistently capitalized

- [ ] **Era-Appropriate Language**
  - [ ] Historical content uses period-appropriate vocabulary
  - [ ] Modern references are absent from historical contexts
  - [ ] Technological terms match timeline position
  - [ ] Cultural references are era-appropriate

- [ ] **Jargon Management**
  - [ ] Technical jargon density matches document purpose
  - [ ] Specialized language is introduced appropriately
  - [ ] Acronyms are defined at first use
  - [ ] Jargon consistency is maintained across related documents

### Grammar & Syntax

- [ ] **Sentence Structure**
  - [ ] Sentence complexity matches content requirements
  - [ ] Syntax patterns are consistent within document type
  - [ ] Variety in sentence structure follows stylistic guidelines
  - [ ] Parallelism is maintained in lists and paired concepts

- [ ] **Paragraph Construction**
  - [ ] Paragraph structure follows consistent pattern
  - [ ] Topic sentences are used appropriately
  - [ ] Transition phrases connect paragraphs smoothly
  - [ ] Paragraph length follows readability standards

- [ ] **Grammatical Conventions**
  - [ ] Tense usage is consistent throughout
  - [ ] Voice (active/passive) follows document standards
  - [ ] Mood (indicative/subjunctive/imperative) is appropriately used
  - [ ] Grammatical person is consistent unless intentionally varied

- [ ] **Punctuation Standards**
  - [ ] Punctuation follows consistent style guide
  - [ ] Special punctuation marks are used appropriately
  - [ ] Quotation mark style is consistent (curly vs. straight)
  - [ ] Terminal punctuation follows document standards

### Stylistic Devices

- [ ] **Literary Techniques**
  - [ ] Metaphor usage is consistent with universe style
  - [ ] Humor devices follow established patterns
  - [ ] Irony/satire levels match document requirements
  - [ ] Rhetorical devices align with content purpose

- [ ] **Emphasis Methods**
  - [ ] Bold and italic formatting follows consistent rules
  - [ ] ALL CAPS usage follows stylistic guidelines
  - [ ] Underline and strikethrough are used appropriately
  - [ ] Emphasis density matches content requirements

- [ ] **Rhythm & Cadence**
  - [ ] Prose rhythm matches content type
  - [ ] Pacing follows document standards
  - [ ] Sentence length variation follows stylistic pattern
  - [ ] Dramatic pauses are implemented consistently

- [ ] **Figurative Language**
  - [ ] Metaphors and similes use universe-appropriate references
  - [ ] Extended metaphors are consistently developed
  - [ ] Symbolism follows established patterns
  - [ ] Personification is used consistently

  - [ ] Emphasis formatting is consistently applied

  - [ ] Bullet points use consistent markers

  - [ ] Parallel structure is maintained within lists

  - [ ] Tables use consistent formatting

  - [ ] Column alignment follows readability standards
  - [ ] Cell formatting is consistent

- [ ] **Special Elements**

  - [ ] Blockquotes follow standard styling
  - [ ] Warning/note blocks use consistent formatting

  - [ ] Hyperlinks follow consistent styling
  - [ ] Internal references use standard format
  - [ ] External links follow security guidelines
  - [ ] Link text is descriptive and consistent

  - [ ] Diagrams use consistent visual language
  - [ ] Legend elements are standardized
  - [ ] Labeling follows documentation standards
  - [ ] Stylistic elements match universe aesthetic

  - [ ] Icons are used consistently for similar concepts

### Accessibility Considerations

- [ ] **Readability Standards**
  - [ ] Text contrast meets accessibility guidelines

  - [ ] Paragraph density balances information and clarity

- [ ] **Navigation Aids**

  - [ ] Navigation links follow consistent pattern

- [ ] **Alternative Content**
  - [ ] Complex content includes simplified explanations
  - [ ] Visual information has text equivalents
  - [ ] Technical content includes appropriate glossaries

- [ ] **Information Hierarchy**
  - [ ] Important information is appropriately highlighted
  - [ ] Content organization follows intuitive patterns
  - [ ] Skimmable elements capture key points
  - [ ] Progressive disclosure is implemented when appropriate
