# Product Visualization Prompt Template
**Edition #1.0.0 | Created: (NEUR-ARC-001) | Last Modified: (NEUR-ARC-001)**

Use this template for creating consistent product visualization prompts across the Turd Bird Universe.

## Template Format

```
# [PRODUCT NAME] - [VISUALIZATION TYPE]
**Style:** Hyperrealistic product photography with technical precision, dramatic lighting, material detail, elegant design with subtle hints of unconventional functionality

**Focus:** [What this specific visualization emphasizes about the product]

**Usage Context:** [Where/how this visualization would be used in documentation]

## Core Prompt
Hyperrealistic [perspective] of [PRODUCT NAME], [brief product description]. [Materials/construction details]. [Design elements]. [Functional features]. [Scale context]. [Usage/demonstration elements]. [Lighting setup]. 8K resolution, photorealistic, product photography lighting, [specific perspective].

## Element Breakdown
- **Product:** [Detailed description of the product's physical form]
- **Materials:** [Key materials, textures, and surface qualities]
- **Features:** [Visible functional elements and distinctive design aspects]
- **Scale:** [Elements that provide size reference]
- **Context:** [Environment or backdrop appropriate to product type]
- **Demonstration:** [How product is shown in use or displaying functionality]
- **Lighting:** [Description of light source, quality, color temperature, and mood]
- **Perspective:** [Viewpoint, distance, angle]

## Variations
1. [Alternative angle/perspective]
2. [Alternative functional state]
3. [Alternative usage context]
4. [Alternative feature highlight]

## References
- [Link to product overview document]
- [Link to technical specifications]
- [Other relevant product documentation]
```

## Example Implementation

```
# Tactical Spork Mark III - Marketing Shot
**Style:** Hyperrealistic product photography with technical precision, dramatic lighting, material detail, elegant design with subtle hints of tactical functionality

**Focus:** Primary dining configuration while suggesting hidden capabilities

**Usage Context:** Main product visualization for Weaponized Kitchenware documentation

## Core Prompt
Hyperrealistic close-up of the Tactical Spork Mark III, the flagship product of Turd Bird Industries' Weaponized Kitchenware line. Elegant utensil crafted from proprietary metal-polymer composite with subtle tactical features. Three-tine design with specialized edge geometry, hollow handle containing concealed tools, bowl section with multi-tool features, base cap with integrated components. Positioned on sleek surface with dramatic product photography lighting. Subtle reflections revealing hidden features. 8K resolution, photorealistic, product photography lighting, three-quarter view with macro detail.

## Element Breakdown
- **Product:** Tactical Spork Mark III, 7.25 inches in length, sophisticated utensil combining fork and spoon functionality with hidden tactical capabilities, elegant design maintaining conventional appearance while incorporating advanced features
- **Materials:** Proprietary metal-polymer composite with matte tactical black finish, subtle seam lines indicating transformable components, specialized edge geometry with micro-serration, varying surface textures between handle and eating surfaces
- **Features:** Three-tine design with specialized geometry, hollow handle with visible access point, bowl section with barely perceptible tool outlines, base cap with integrated fire-starting component, activation points subtly indicated through design elements
- **Scale:** Positioned next to standard dining elements for size comparison, subtle measurement indicators etched into handle
- **Context:** Premium product display environment with sleek dark surface creating reflections, suggestion of high-end dining context while hinting at tactical versatility
- **Demonstration:** Primary dining configuration shown, but positioned to suggest transformability, subtle indications of alternate configurations visible on close inspection
- **Lighting:** Dramatic product photography lighting with main key light creating strong highlights on edges, rim lighting outlining form, controlled reflections revealing hidden features
- **Perspective:** Three-quarter view showing both profile and front aspects, macro lens detail revealing precision engineering, shallow depth of field focusing on key features

## Variations
1. Overhead view showing all components separately laid out in technical arrangement
2. Action shot showing transformation sequence in motion blur
3. Side-by-side comparison with conventional utensils highlighting enhanced capabilities
4. Tactical deployment view showing all extended tools and features
```
