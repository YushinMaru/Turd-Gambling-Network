## Usage Instructions

### Direct Copy-Paste Usage
1. Open the desired prompt file
2. Copy the entire text (no need to modify)
3. Paste directly into AI image generation tool
4. Generate image

### Customization Guidelines
If modifications are needed:
1. Copy the original prompt as a base
2. Modify specific elements while maintaining structure
3. Preserve all key character/location details for consistency
4. Maintain the extreme level of detail
5. Keep technical specifications at end of prompt

## Generation Requirements
All Turd Bird Universe AI image prompts:
- Maintain extreme detail for maximum fidelity
- Include comprehensive physical descriptions
- Specify precise environmental context
- Detail lighting and atmospheric conditions
- Maintain narrative consistency with established universe
- Specify technical parameters for optimal generation
